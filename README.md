# World Cup Stock Exchange dashboard

This version uses the editable Google Sheet ID and loads the public tabs by sheet name using Google's Visualization API JSONP endpoint.

## Important Google Sheet settings

1. Open the Google Sheet.
2. Click Share.
3. Set General access to **Anyone with the link**.
4. Role: **Viewer**.

The sheet does not need to be editable by the public.

## Upload

Replace your old `index.html` in GitHub with this one, commit, then hard refresh the GitHub Pages site with Ctrl + F5.
